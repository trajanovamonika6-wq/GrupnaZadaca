## Опис
Систем за евиденција на присуство во училница преку NFC технологија.

## Компоненти
- **TeacherApp** - Android апликација за наставник (прима NFC, синкронизира со сервер)
- **StudentApp** - Android апликација за студент (праќа NFC)
- **cps** - Backend API (PHP + MySQL) и Web Dashboard

## Технологии
- Android Studio + Java
- PHP + MySQL (XAMPP)
- Room база, Retrofit, NFC/HCE
